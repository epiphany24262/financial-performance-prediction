## 2026-06-09: Group identical feature records for OOF

- Group IDs are built from non-target model features after converting `+/-inf` to `NaN`.
- All current B0-B4 baselines use `GroupKFold(n_splits=5)`.
- `tests/test_cv_no_leakage.py` verifies that groups do not cross train/validation folds.

## 2026-06-09: Install only immediate test dependencies

- Installed: `pytest`, `pytest-cov`, `catboost`.
- Not installed: `xgboost`, `lightgbm`, `optuna`, `jupyter`.
- Environment audit and lock files were regenerated after installation.

## 2026-06-09: Select B4 as current best baseline

- B4 is an OOF-selected per-target blend of B1/B2/B3 rule predictions.
- Weight selection used only GroupKFold OOF predictions and training targets.
- Current B4 OOF mean R2: `0.7830834896750798`.
- The best baseline is not the final model; `Q0_TOTAL_STOCKHOLDERS_EQUITY` remains weak.

## 2026-06-09: Run M1/M2 on raw historical features before CatBoost

- M1 Ridge uses fold-fitted median imputation, RobustScaler, and Ridge `solver="svd"` for numerical stability.
- M2 HistGradientBoosting uses fold-fitted preprocessing and the same GroupKFold split.
- Both models use `history_raw` features only, matching the ablation table's raw-history sklearn baselines.
- Results:
  - M1 Ridge OOF mean R2: `0.7313902644659322`
  - M2 HistGradientBoosting OOF mean R2: `0.4971715803852488`
- Decision: keep B4 as the current best average-score model and move to CatBoost direct prediction next.

## 2026-06-09: Use CatBoost direct and residual runs as the next best candidate

- CatBoost direct on `history_metadata_engineered` (now tracked as M3d) reached OOF mean R2 `0.6090404573760977`.
- CatBoost residual on the same feature set reached OOF mean R2 `0.8252497284371879`, which beats B4.
- The residual result is now the current best candidate for downstream fusion and final submission assembly.

## 2026-06-09: Allow selective CatBoost experiment runs for reruns

- `scripts/run_catboost_models.py` accepts `CATBOOST_EXPERIMENT_IDS` so a single experiment can be rerun without recomputing the full CatBoost block.
- It also accepts experiment IDs as command-line arguments for reproducible one-off reruns.
- The default behavior still runs the full CatBoost experiment set when no selector is provided.

## 2026-06-10: Complete CatBoost metadata ablation

- M3a uses raw historical features only and reached OOF mean R2 `0.554307`.
- M3b adds `industry` and `sector` and reached OOF mean R2 `0.569533`.
- M3c adds all metadata and reached OOF mean R2 `0.608393`.
- M3d adds engineered time-series, change, ratio, missingness, and signed-log features and reached OOF mean R2 `0.6090404573760977`.
- Decision: keep M3d as the direct CatBoost member for final OOF blending.

## 2026-06-09: Select M6 OOF blend as the final model

- M6 blends B4, M3d, and M4 separately for each target using OOF R2 only.
- Final OOF mean R2: `0.8572266051577525`.
- Best config now points to `M6_oof_blend`.
- The final submission preserves `sample_submission.csv` column order and Id order.
